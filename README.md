## `[ 01 ] // ORIGIN STORY`

> Every hacker has a moment. Mine was simple —  
> I ran my first Nmap scan and the world opened up.  
> Ports. Protocols. Services. All just *waiting* to be understood.

Since then it's been a rabbit hole of:
- **Recon** — mapping attack surfaces before anyone notices
- **Bug Bounty** — turning curiosity into responsible disclosure
- **CTF** — competing, failing, learning, competing again
- **Networking** — because you can't break what you don't understand

The mission isn't destruction. It's *comprehension*.
